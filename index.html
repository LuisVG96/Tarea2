¿Qué es HTML?
HTML es el lenguaje estándar para crear páginas web. Define la estructura del contenido mediante etiquetas.
¿Qué es CSS?
CSS es un lenguaje que controla la presentación visual de documentos HTML. Maneja aspectos como colores, fuentes y diseños.
¿Qué es JavaScript?
JavaScript es un lenguaje de programación que permite agregar interactividad y dinamismo a las páginas web.
¿Quién pintó «La noche estrellada»?
«La noche estrellada» fue pintada por Vincent van Gogh en 1889. Es una de sus obras más famosas del postimpresionismo.
¿Qué es la fotosíntesis?
Proceso mediante el cual las plantas convierten luz solar, CO₂ y agua en glucosa (energía) y oxígeno, usando clorofila.
¿Quién escribió «Cien años de soledad»?
Gabriel García Márquez, escritor colombiano y premio Nobel de Literatura en 1982.
